# Technology Stack

- **Backend**: Kotlin with Spring Boot
- **Database**: MongoDB
- **Frontend**: React/Remix (keruta-admin)
- **Task Execution**: Go (keruta-agent)
- **Protocol**: Kotlin Multiplatform (KTCP for WebSocket communication)
- **Build Tool**: Gradle with Kotlin DSL
- **Testing**: JUnit 5, TestContainers, Mockito
- **Code Quality**: ktlint for Kotlin formatting
- **Containerization**: Docker
- **Authentication**: Keycloak with JWT tokens
- **Workspace Management**: Coder integration
